- # Invoice Generator
- A full-stack **AI-powered invoice generator** that allows users to create, manage, and analyze invoices with automated insights and email reminders.

- ## Features
- Secure user authentication
- Create, edit, and delete invoices (CRUD)
- AI-powered invoice creation & insights
- PDF invoice generation
- Automated email reminders for due invoices
- Invoice analytics & status tracking

- ## Tech Stack
- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **AI:** Google Gen AI 
- **Email:** Nodemailer  
- **Deployment:**  
  - Frontend → Vercel  
  - Backend → Render
